A little Description to describe the plugin

showup.js

An image caption compiler

  • github
  • Examples

Install

git clone git://github.com/DanielTamkin/shimThis.js.git

showup.js is a stupid simple plugin that solves a problem i found with ghost, you can't caption photos on your post. This handy-dandy jQuery reliant plugin fixes that. Just plop in this bit of code into your image code, like so:

<img alt="!showup//say whatever you want!//!a cat gif" src="cat.gif">

notice the "!showup//" and "//!" in the alt tag, these are starting and ending tags for showup, anything between these tags will be your caption. Once you've done that, lets get you running the compiler.
Its super easy, target the element that will have the <img> in it. Like this: 

  <script>
  $("section.images-here").showup();
  </script>

Awesome, now when you run the page the line where you have you image will be replaced with this: 

  <div class="showup-parent">
    <img alt="a cat gif" src="cat.gif">
    <p>say whatever you want!</p>
  </div>

There you go! a super simple complier to get some cool captions out of your work.

ps...

if your like me and want to have captions for say ghost, or anything to do with markdown. The code is the exact same, so there is no way to screw up.

![!showup//say whatever you want!//!a cat gif](cat.gif)

A dog photo

Do not fear.

If the compiler finds an image with no showup code, do not fear! it will not compile it, so code like:

<img alt="a dog gif" src="dog.gif">

will be un-touched.

Also,

if the compiler detects that you didnt end the showup code( as you should with //!) The beginning tag(showup!//) will be taken out, so to recover your alt tag's data as much as it can.
